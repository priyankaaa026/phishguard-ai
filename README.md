# 🔐 PhishGuard AI

AI-powered cybersecurity tool that detects phishing messages, malicious content, and suspicious URLs using machine learning.

---

## 🚀 Features

- Detects phishing and malicious messages  
- Real-time prediction  
- Step-by-step AI decision process (agent-like behavior)  
- Simple and interactive UI  

---

## 🧠 How it works

1. User enters message or URL  
2. Text is converted into numerical features using TF-IDF  
3. Machine learning model (Naive Bayes) analyzes input  
4. System classifies content as Safe or Malicious  
5. Displays result with explanation  

---

## ⚙️ Tech Stack

- Python  
- scikit-learn  
- Pandas  
- NumPy  
- Streamlit  
- TF-IDF  
- Naive Bayes  

---

## 📊 Model Accuracy

~90% accuracy on test dataset  

---

## ▶️ How to Run

1. Clone the repository  

git clone https://github.com/priyankaaa026/phishguard-ai.git


2. Install dependencies  

pip install pandas numpy scikit-learn streamlit


3. Train the model  

python model.py


4. Run the app  

streamlit run app.py
